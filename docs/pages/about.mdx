## 订阅与专业功能

许多个人和组织赞助我们的工作。如果您正在使用 BlockNote，支持该项目的最简单方式是订阅 BlockNote Pro。订阅还可获得核心团队的 **优先支持**。

订阅 BlockNote Pro 可让您访问专业示例代码，并获得 BlockNote 团队的额外支持。更多有关 BlockNote Pro 订阅信息，请访问[定价页面](/pricing)。

我们的赞助商之一是著名的 NLNet 基金会，其使命是支持开放互联网，保护互联网用户的隐私和安全。[了解他们](https://www.nlnet.nl)。

BlockNote 构建在 [ProseMirror](https://prosemirror.net/) 之上——这是另一个值得支持的优秀项目。

## 贡献

另一种重要贡献方式是成为 Discord 社区成员，帮助回答社区问题，并在 GitHub 上贡献代码库和文档。
社区还从[示例](/examples)中大受裨益，欢迎分享有用的代码片段！

此外，查看完整贡献者列表请访问 [GitHub](https://github.com/TypeCellOS/BlockNote/graphs/contributors)。
