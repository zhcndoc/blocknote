# 编辑器 API

BlockNote 提供了一个 API，用于通过代码与编辑器及其内容进行交互。
这些方法可直接在你创建编辑器实例时得到的 `BlockNoteEditor` 对象上使用。

- [操作区块](/docs/editor-api/manipulating-blocks.md) 讲解如何读取/更新文档中的区块。
- [操作内联内容](/docs/editor-api/manipulating-inline-content.md) 讲解如何更新/读取选中文本的数据。
- [光标与选择](/docs/editor-api/cursor-selections) 讲解与光标位置和选择相关的方法。
- [Markdown 与 HTML](/docs/converting-blocks) 讲解如何将文档转换为 Markdown 和 HTML，以及从它们转换回来。
