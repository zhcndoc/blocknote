# 添加 DOM 属性

您可以在编辑器内的多个 DOM 元素上设置额外的 HTML 属性。这些元素包括：

`editor:` 编辑器本身，不包括菜单和工具栏。

`block:` 块的主容器元素。包含块的内容及其嵌套块。

`blockGroup:` 编辑器中所有顶级块和嵌套块的包装元素。

`blockContent:` 块内容的包装元素。

`inlineContent:` 块富文本内容的包装元素。
