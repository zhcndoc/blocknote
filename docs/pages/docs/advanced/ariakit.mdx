## 使用 Ariakit 与 BlockNote

[Ariakit](https://ariakit.org/) 是一个开源的无样式（headless）、基础组件库，注重无障碍性。要将 BlockNote 与 Ariakit 一起使用，可以从 `@blocknote/ariakit` 中导入 `BlockNoteView`（而不是从 `@blocknote/mantine` 导入）。

你可以使用自己的 CSS 完全自定义样式，或者通过导入 `@blocknote/ariakit/style.css` 中提供的默认样式文件。
