# 样式与主题

你可以完全改变 BlockNote 编辑器的外观和感觉。使用 [主题 CSS 变量](/docs/styling-theming/themes) 快速更改基本样式，或通过 [附加 CSS 规则](/docs/styling-theming/overriding-css) 应用更复杂的样式。

如果你想更改、移除或完全替换菜单和工具栏的 React 组件，请参见 [UI 组件](/docs/ui-components)。
