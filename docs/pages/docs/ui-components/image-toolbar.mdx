# 图片工具栏

当你选择一个没有 URL 的图片，或者在选择图片时点击 [格式工具栏](/docs/formatting-toolbar)中的“替换图片”按钮时，图片工具栏会出现。

## 图片上传

你可能会注意到，在新建 BlockNote 编辑器时，图片工具栏中的“上传”标签是缺失的。这是因为你必须为 BlockNote 提供一个处理文件上传的函数，使用 `uploadFile` [编辑器选项](/docs/editor)：

```ts
type uploadFile = (file: File) => Promise<string>;
```

`file:` 要上传的文件，此处指图片。

`returns:` 一个 `Promise`，解析后返回图片可访问的 URL。

你可以使用提供的 `uploadToTempFilesOrg` 函数作为起点，该函数将文件上传到 [tmpfiles.org](https://tmpfiles.org/)。但不建议在生产环境中使用它，你应该使用自己的后端：
