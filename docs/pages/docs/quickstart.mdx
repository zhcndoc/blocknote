# 快速开始

使用 BlockNote 快速且轻松地入门。安装所需的包并将 React 组件添加到您的应用中。

## 使用 NPM 安装

要通过 [NPM](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) 安装 BlockNote，请运行：

```console
npm install @blocknote/core @blocknote/react @blocknote/mantine
```

## 创建编辑器

使用 `useCreateBlockNote` 钩子，我们可以创建一个新的编辑器实例，然后使用 `BlockNoteView` 组件进行渲染。见下方示例：

<Example name="basic/minimal" />

我们还导入 `@blocknote/mantine/style.css`，为编辑器添加默认样式以及 BlockNote 提供的 `Inter` 字体（可选）。

## 接下来

现在您已经知道如何将 BlockNote 集成到您的 React 应用中！不过，这只是 BlockNote 功能的冰山一角。

- 了解 [区块与编辑器基础](/docs/editor-basics) 以及如何使用 [编辑器 API](/docs/editor-api) 与编辑器交互
- 查看 [UI 组件](/docs/ui-components) 来定制内置菜单和工具栏，以及 [样式与主题](/docs/styling-theming) 来自定义编辑器的外观和感觉
- 使用 [自定义架构](/docs/custom-schemas) 扩展编辑器，或添加 [实时协作](/docs/advanced/real-time-collaboration) 功能
